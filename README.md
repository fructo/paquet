## 🔨 Tasks

To cache all required dependencies:

```
deno task cache
```

To run the app in development mode:

```
deno task dev
```

To update apps:

```
deno task update [appId] [--force]
```

To manage dependencies:

```
deno task esm:[add|remove|update]
```

To type check, lint, and format the source code:

```
deno task check
```
